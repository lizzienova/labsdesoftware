2. O objetivo era verificar se a quantidade de bits 1 de um inteiro é ímpar, sem usar %. A solução foi usar a operação XOR, pois XOR de todos os bits resulta 1 se o número de bits 1 for ímpar
exemplo:
    Para 0x01010101 → número par de bits 1 → retorna 0
    Para 0x01030101 → número ímpar → retorna 1
